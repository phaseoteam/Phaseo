# Apps Directory

This folder holds the three main parts of **Phaseo** — each a core piece of the ecosystem.

| App              | Description                         |
| ---------------- | ----------------------------------- |
| [`web`](./web)   | The public website and dashboard.   |
| [`docs`](./docs) | The Mintlify-powered documentation. |
| [`api`](./api)   | The Cloudflare-based API gateway.   |

Each can run independently, but they’re designed to work together — forming a complete open infrastructure for model tracking and access.

---

### 🧭 Overview

-   **`web/`** - The user interface for exploring AI models, releases, and benchmarks.
-   **`api/`** - The backend that connects to every AI provider through one standard interface.
-   **`docs/`** - The documentation that explains how everything fits together and how to contribute.

---

### 🤝 Contributing

To work on a specific app, go into its folder and read its README — each has guidance on structure and contribution.  
You can also make changes across multiple apps if you’re improving shared utilities or adding new data.

> All development happens in public. PRs and discussions are always welcome.
