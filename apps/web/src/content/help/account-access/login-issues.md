If you cannot sign in, run through this checklist:

## Basic checks

- Confirm you are using the same sign-in method you used when creating your account.
- Re-enter your email manually to avoid autofill mistakes.
- Make sure Caps Lock is not enabled.

## If your password fails

1. Use **Forgot password** on the sign-in page.
2. Open the latest reset email and complete the flow from that message.
3. If you requested multiple reset emails, only use the newest link.

## If you still cannot access your account

- Clear browser cache/cookies for the site and try again.
- Try an incognito/private window to rule out extension conflicts.
- Contact support with:
  - the email tied to the account
  - the exact error message
  - approximate time the issue occurred
