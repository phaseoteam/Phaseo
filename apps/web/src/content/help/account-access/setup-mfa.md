MFA adds a second security step after your password and is strongly recommended.

## Before you begin

- Install an authenticator app (for example Google Authenticator, 1Password, or Authy).
- Make sure your device time is set automatically.

## Setup steps

1. Go to **Settings > Account > Security**.
2. Start MFA enrollment.
3. Scan the QR code in your authenticator app.
4. Enter the 6-digit verification code.
5. Confirm enrollment.

## If codes are not accepted

- Wait for a new 30-second code window and try again.
- Verify your phone time is synced automatically.
- Re-enroll MFA if the problem continues.
