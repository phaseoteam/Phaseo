Use this flow if you forgot your password:

1. Go to the sign-in page.
2. Select **Forgot password**.
3. Enter your account email and submit.
4. Open the password reset email and follow the link.
5. Set a new password and sign in again.

## Troubleshooting reset emails

- Check spam/junk folders.
- Wait a few minutes and request a new email.
- If you requested multiple links, only the latest one is valid.

## Password guidance

- Use a unique password for this account.
- Avoid reusing passwords from other services.
- Store your password in a password manager.
