March was a foundational release month for API coverage and model breadth.

This recap covers shipping from **March 1, 2026** through **March 31, 2026**.

## The Big Drops

### Embeddings API support expanded

We significantly expanded embeddings support across the stack, including API surface updates, docs coverage, SDK examples, and product surfaces for embeddings workflows.

Highlights included:

- Embeddings endpoint and schema updates across API and OpenAPI
- SDK model surface and examples updates for embeddings
- Dedicated embeddings experience in chat and provider model views
- Added support for additional embedding models, including Gemini embedding updates

- [/models](/models)

### Multimodal support moved forward

March also pushed multimodal coverage forward across audio and video pathways, with related executor, routing, and docs updates landing through the month.

### Major model coverage updates

We shipped broad model and pricing updates, including notable additions and upgrades such as:

- Claude Opus 4.6 and Sonnet 4.6 coverage
- GPT-5.4 Mini and Nano data/docs updates
- GLM 5 Turbo and new Mistral March model onboarding
- Gemini 3.1 Flash Lite onboarding and benchmark refreshes
- Grok 4.20 routing/data sync updates

### SDK and schema alignment

We shipped coordinated schema and SDK alignment work so model and capability changes landed consistently across generated SDKs and docs.
