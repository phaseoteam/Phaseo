April moved fast, and we shipped a lot.

Welcome to the first public **ICYMI**: a quick recap of the biggest updates we shipped between **April 1, 2026** and **April 30, 2026**.

## The Big Drops

### New pricing insights

We upgraded pricing across model pages with clearer pricing history and stronger pricing insights.

Try it in product:

- [/pricing](/pricing)
- [/tools/pricing-calculator](/tools/pricing-calculator)

### Stronger gateway and SDK surface

April included major gateway and SDK progress: full API rollout work, rerank support, and active-model gating across API and SDKs.

The result is a more consistent experience across docs, API behavior, and SDK clients.

### Model coverage jumped

We shipped Claude Opus 4.7 compatibility and expanded provider coverage, including deeper Anthropic Bedrock/Vertex mappings and pricing updates.

### Catalog updates kept landing

We added and enriched support for major models, including Qwen 3.6 35B A3B, Grok 4.3, and other catalog improvements throughout the month.

Explore it:

- [/models](/models)
- [/updates](/updates)

### Faster model pages and better updates UX

We improved model page performance by reducing fetch fanout and refining cache behavior, plus updates UX improvements like clearer calendar release-day drilldowns.

## Under the Hood Hardening

Alongside features, we shipped production hardening across billing, auth, cache, and data paths:

- Stripe and credits flow fixes
- Auth disclosure hardening
- Supabase/cache egress reductions
- Free-model gating and pricing-rule corrections
