---
title: "Model endpoints (Beta)"
badge: "Beta"
openapi: "GET /models/{author}/{slug}/endpoints"
description: "List provider-specific endpoint capabilities for one model."
---

Use this endpoint when you already know a model ID and need the provider-by-provider capability table behind it.

Each row describes one provider and endpoint pairing:

- `endpoint` and `capability_id`
- `public_path`
- `collection`
- `provider_id`, `provider_name`, and `provider_model_slug`
- `input_modalities` and `output_modalities`
- `supported_parameters` and `supported_parameters_detail`
- provider, model, and capability availability/routing states
- `pricing` and `pricing_detail`

```bash
curl "https://api.phaseo.app/v1/models/openai/sora/endpoints?supported_parameters=duration" \
  -H "Authorization: Bearer $AI_STATS_API_KEY"
```
