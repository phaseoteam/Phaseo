Provisioning keys provide elevated privileges for programmatic management of your AI Stats resources. Use provisioning keys to list, manage, and delete keys for your team.

## Authentication

All endpoints require a provisioning key in the Authorization header:

```http
Authorization: Bearer <your-provisioning-key>
```
