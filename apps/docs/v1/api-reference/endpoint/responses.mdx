## Server tools

`/v1/responses` supports regular function tools and these gateway-managed server tools:

- `gateway:datetime`
- `ai-stats:web_search`
- `ai-stats:web_fetch`

### Datetime example

```bash
curl https://api.phaseo.app/v1/responses \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/gpt-5-nano",
    "stream": false,
    "input": [
      { "role": "user", "content": "Give me the current datetime in Asia/Tokyo." }
    ],
    "tools": [
      {
        "type": "gateway:datetime",
        "parameters": { "timezone": "Asia/Tokyo" }
      }
    ],
    "tool_choice": "auto"
  }'
```

When a server tool is used, usage includes the matching `usage.server_tool_use.*` counters.

- `gateway:datetime` -> `datetime_requests`
- `ai-stats:web_search` -> `web_search_requests`
- `ai-stats:web_fetch` -> `web_fetch_requests`

## Streaming note

`stream: true` is also supported. For gateway-managed server tools, AI Stats may materialize the upstream tool-call turn and re-emit a synthetic stream after continuing the loop.
