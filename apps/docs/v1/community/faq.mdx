## Where do I get an API key?

Create keys in the AI Stats dashboard. See [Authentication](/v1/developers/authentication).

## What base URL should I use?

Use `https://api.phaseo.app/v1` as the base URL for the Gateway.

## Which endpoint should I start with?

Use `/v1/responses` for most text workflows. `/v1/chat/completions` and `/v1/messages` are available for compatibility.

## How do I see supported models?

Use the [Models endpoint](/v1/api-reference/endpoint/models) for the latest list.

## How does BYOK work?

Bring-your-own-key lets you connect provider keys while still using the Gateway. See [Provisioning keys](/v1/developers/provisioning-keys).

## Where can I ask for help?

See [Contact and support](/v1/community/contact-and-support) for help channels.
