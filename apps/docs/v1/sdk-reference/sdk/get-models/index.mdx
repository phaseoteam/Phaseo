Use the SDKs to call `/gateway/models`. The response mirrors the live gateway catalogue: public model IDs, public endpoint names like `chat.completions`, provider availability, supported params, and pricing.

<CodeGroup dropdown>
```ts TypeScript
import { AIStats } from "@ai-stats/ts-sdk";

const client = new AIStats({ apiKey: process.env.AI_STATS_API_KEY! });
const res = await client.getModels({ limit: 5, privacy_scope: "shared" });
console.log(res.models);
````

```python Python
from ai_stats import AIStats

client = AIStats(api_key="YOUR_KEY")
resp = client.get_models({"limit": 5, "privacy_scope": "shared"})
print(resp.models)
````
