---
title: "Generate Text (AI SDK)"
description: "Non-streaming text generation with Vercel AI SDK and Phaseo."
---

Use `generateText` with `phaseo(modelId)` for one-shot completions.

```ts
import { phaseo } from "@phaseo/ai-sdk-provider";
import { generateText } from "ai";

const result = await generateText({
  model: phaseo("openai/gpt-5-nano"),
  prompt: "Write a concise summary of Phaseo Gateway.",
});

console.log(result.text);
console.log(result.providerMetadata);
```

## With chat history

```ts
import { phaseo } from "@phaseo/ai-sdk-provider";
import { generateText } from "ai";

const result = await generateText({
  model: phaseo("openai/gpt-5-nano"),
  messages: [
    { role: "system", content: "You are a terse assistant." },
    { role: "user", content: "Give me three bullet points on retrieval augmentation." },
  ],
});

console.log(result.text);
console.log(result.providerMetadata);
```

## Per-model settings

```ts
import { phaseo } from "@phaseo/ai-sdk-provider";
import { generateText } from "ai";

const result = await generateText({
  model: phaseo("openai/gpt-5-nano", {
    temperature: 0.2,
    topP: 0.9,
    maxTokens: 300,
  }),
  prompt: "Draft a release note for an API update.",
});

console.log(result.text);
console.log(result.providerMetadata);
```

## Prompt vs messages

- Use `prompt` for quick single-turn requests.
- Use `messages` for multi-turn context and role control.
- Keep model IDs dynamic by querying `/v1/models` (see [Model Discovery](/v1/sdk-reference/sdk/get-models)).
