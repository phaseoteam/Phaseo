---
title: "Stream Text (AI SDK)"
description: "Streaming text generation with Vercel AI SDK and Phaseo."
---

Use `streamText` when you want partial tokens as they arrive.

```ts
import { phaseo } from "@phaseo/ai-sdk-provider";
import { streamText } from "ai";

const result = streamText({
  model: phaseo("openai/gpt-5-nano"),
  prompt: "Explain SSE streaming in plain language.",
});

for await (const chunk of result.textStream) {
  process.stdout.write(chunk);
}

console.log(await result.providerMetadata);
```

## Streaming with messages

```ts
import { phaseo } from "@phaseo/ai-sdk-provider";
import { streamText } from "ai";

const result = streamText({
  model: phaseo("openai/gpt-5-nano"),
  messages: [
    { role: "user", content: "Write a five-step rollout plan for a gateway migration." },
  ],
});

let text = "";
for await (const chunk of result.textStream) {
  text += chunk;
}

console.log(text);
console.log(await result.providerMetadata);
```

## Production tips

- Prefer streaming for long responses and interactive UIs.
- Handle cancellation/timeout in your runtime for abandoned streams.
- Keep model IDs fresh via `/v1/models` to avoid stale aliases.
