---
title: "Moderations"
description: "Call /moderations with the Ruby SDK."
---

**Methods**: `client.generate_moderation(...)`, `client.create_moderation(...)`.

### Example
```ruby
require 'phaseo_sdk'

client = PhaseoSdk::Phaseo.new(api_key: ENV.fetch("PHASEO_API_KEY"))

moderation = client.create_moderation(
  "model" => "openai/omni-moderation-latest",
  "input" => "Text to classify"
)
```

### Key parameters
- `model` (required): Moderation model id (e.g., `openai/omni-moderation-latest`).
- `input` (required): String or array of content items (text or image_url).
- Optional: `categories`/`severity` filters (when supported), `user` tag.

### Returns
`ModerationResponse`
