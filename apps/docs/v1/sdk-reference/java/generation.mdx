---
title: "Generation lookup"
description: "Call /generations to retrieve a prior generation record."
---

**Method**: `Operations.getGeneration(...)`

```java
import ai.stats.gen.Operations;
import java.util.HashMap;
import java.util.Map;

Map<String, String> query = new HashMap<>();
query.put("id", "G-01ABC123");

Object generation = Operations.getGeneration(client, null, query, null, null);
System.out.println(String.valueOf(generation));
```
