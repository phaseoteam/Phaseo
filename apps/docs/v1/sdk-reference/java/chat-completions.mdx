---
title: "Chat completions"
description: "Call /chat/completions with the Java SDK."
---

**Method**: `client.createChatCompletion(...)`

```java
import app.phaseo.sdk.Phaseo;

Phaseo client = new Phaseo(System.getenv("PHASEO_API_KEY"));

String body = """
{
  "model": "openai/gpt-5-nano",
  "messages": [
    { "role": "user", "content": "Write a one-line haiku." }
  ]
}
""";

Object response = client.createChatCompletion(body);
System.out.println(String.valueOf(response));
```
