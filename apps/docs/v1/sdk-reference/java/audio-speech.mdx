---
title: "Audio speech"
description: "Call /audio/speech with the Java SDK."
---

**Method**: `Operations.createSpeech(...)`

```java
import ai.stats.gen.Operations;

String body = """
{
  "model": "openai/gpt-4o-mini-tts",
  "input": "Hello from Java",
  "voice": "alloy",
  "format": "mp3"
}
""";

Object response = Operations.createSpeech(client, null, null, null, body);
System.out.println(String.valueOf(response));
```
