---
title: "STT (Speech to Text)"
description: "Call /audio/transcriptions with the Java SDK."
---

**Method**: `client.createTranscription(...)`

```java
import app.phaseo.sdk.Phaseo;

Phaseo client = new Phaseo(System.getenv("PHASEO_API_KEY"));

String body = """
{
  "model": "openai/whisper-1",
  "file": "data:audio/mpeg;base64,..."
}
""";

Object response = client.createTranscription(body);
System.out.println(String.valueOf(response));
```
