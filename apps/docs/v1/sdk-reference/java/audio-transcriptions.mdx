---
title: "Audio transcriptions"
description: "Call /audio/transcriptions with the Java SDK."
---

**Method**: `Operations.createTranscription(...)`

```java
import ai.stats.gen.Operations;

String body = """
{
  "model": "openai/whisper-1",
  "file": "data:audio/mpeg;base64,..."
}
""";

Object response = Operations.createTranscription(client, null, null, null, body);
System.out.println(String.valueOf(response));
```
