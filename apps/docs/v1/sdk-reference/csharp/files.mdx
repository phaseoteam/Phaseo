---
title: "Files"
description: "Use the C# SDK for file upload, metadata, and file-content retrieval."
---

**Methods**: `client.UploadFile()`, `client.ListFiles()`, `client.RetrieveFile()`, `client.RetrieveFileContent()`.

### Example
```csharp
var client = new Phaseo(apiKey: apiKey);

var uploaded = await client.UploadFile(new Dictionary<string, object>
{
    ["purpose"] = "batch",
    ["file"] = "data:application/json;base64,eyJ0ZXN0Ijp0cnVlfQ=="
});

var file = await client.RetrieveFile("file_123");
var content = await client.RetrieveFileContent("file_123");
```

<Note>
`ListFiles()` is wired through the SDK, but `GET /files` currently returns `file_list_not_supported_with_shared_gateway_key`
on the shared gateway key. Persist uploaded file ids and retrieve them directly instead.
</Note>
