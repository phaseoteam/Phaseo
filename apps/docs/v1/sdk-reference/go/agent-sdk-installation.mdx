## Install

```bash
go get github.com/phaseoteam/Phaseo/packages/sdk/agent-sdk-go@latest
```

## Environment

Set your gateway API key before running examples:

```bash
export PHASEO_API_KEY="phaseo_v1_sk_..."
```
