---
title: "Image edits"
description: "Edit images with the Go SDK."
---

Use the wrapper method:

- `client.CreateImageEdit(ctx, req)`

```go
import (
  "context"
  "fmt"
  phaseo "github.com/phaseoteam/Phaseo/packages/sdk/sdk-go"
)

client := phaseo.New(apiKey, "https://api.phaseo.app/v1")

resp, err := client.CreateImageEdit(context.Background(), map[string]interface{}{
  "model": "openai/gpt-image-1",
  "prompt": "Turn this into a watercolor style",
  "image": "data:image/png;base64,...",
})
if err != nil {
  panic(err)
}

fmt.Println(resp)
```
