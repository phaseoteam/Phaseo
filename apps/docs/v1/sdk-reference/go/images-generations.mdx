---
title: "Image generations"
description: "Generate images with the Go SDK."
---

Use the wrapper method:

- `client.CreateImage(ctx, req)`

```go
import (
  "context"
  "fmt"
  phaseo "github.com/phaseoteam/Phaseo/packages/sdk/sdk-go"
)

client := phaseo.New(apiKey, "https://api.phaseo.app/v1")

resp, err := client.CreateImage(context.Background(), map[string]interface{}{
  "model": "openai/gpt-image-1",
  "prompt": "A lighthouse at sunset",
})
if err != nil {
  panic(err)
}

fmt.Println(resp)
```
