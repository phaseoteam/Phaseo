---
title: "STT (Speech to Text)"
description: "Call /audio/transcriptions with the Go wrapper."
---

Use the wrapper method:

- `client.CreateTranscription(ctx, req)`

```go
import (
  "context"
  "fmt"
  phaseo "github.com/phaseoteam/Phaseo/packages/sdk/sdk-go"
)

client := phaseo.New(apiKey, "https://api.phaseo.app/v1")
resp, err := client.CreateTranscription(context.Background(), map[string]interface{}{
  "model": "openai/gpt-4o-transcribe",
  "audio_b64": "data:audio/wav;base64,...",
  "language": "en",
})
if err != nil {
  panic(err)
}
fmt.Println(resp)
```
