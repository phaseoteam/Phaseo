---
title: "Embeddings"
description: "Call /embeddings with the Go wrapper."
---

Use the wrapper method:

- `client.GenerateEmbedding(ctx, req)`

```go
import (
  "context"
  "fmt"
  phaseo "github.com/phaseoteam/Phaseo/packages/sdk/sdk-go"
)

client := phaseo.New(apiKey, "https://api.phaseo.app/v1")
resp, err := client.GenerateEmbedding(context.Background(), map[string]interface{}{
  "model": "openai/text-embedding-3-large",
  "input": "sample text",
})
if err != nil {
  panic(err)
}

fmt.Println(resp)
```
