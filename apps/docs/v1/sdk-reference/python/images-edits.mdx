---
title: "Image edits"
description: "Call /images/edits with the Python SDK."
---

**Method**: `client.generate_image_edit()`

```python
result = client.generate_image_edit({
    "model": "openai/gpt-image-1",
    "prompt": "Make it sunset",
    "image": "data:image/png;base64,...",
})

print(result)
```
