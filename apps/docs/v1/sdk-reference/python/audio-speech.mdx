---
title: "TTS (Text to Speech)"
description: "Call /audio/speech with the Python SDK."
---

**Method**: `client.generate_speech()`

```python
result = client.generate_speech({
    "model": "openai/gpt-4o-mini-tts",
    "input": "Hello world",
    "voice": "alloy",
    "format": "mp3",
})

print(result)
```
