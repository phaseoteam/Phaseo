# AI Stats Agent SDK (Ruby)

`ai_stats_agent_sdk` is a minimal Ruby agent runtime for AI Stats Gateway.

It provides:

- `AIStatsAgentSdk.create_agent`
- `AIStatsAgentSdk.define_tool`
- `AIStatsAgentSdk.create_gateway_agent_client`
- a bounded tool loop on top of the AI Stats `responses` API

## Install

```bash
gem install ai_stats_sdk ai_stats_agent_sdk
```
